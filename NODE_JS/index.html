Using Command Line in the Terminal with Node

  • pwd (print working directory) - to show the current path in the directory
  • cd (change directory) - to change to another directory by specifying the name of that directory
  • ls - to list the files in a certaindirectory
  • mkdir (make directory) - to create a new directory or folder followed by the name to be given to the folder
  • touch - to create a new file. It is followed by specifying the name of the folder

REPL

This stands for Read Evaluation Print Loop. And it allows to execute codes little by little.

This is basically a way to write and execute Javascript code in the terminal using Node Js. By typinf node in the terminal, the REPL would be initiated.
